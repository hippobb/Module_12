The module 12 challenge is a SQL HR program. It can be able to view and manage the departments, roles, and employees in the program.
It is a command-line application that accepts user input. WHEN the application startup a main menu will be display for the user to select view, add, or update. After selected, it will show the sub-menu accordingly. 
In the view menu, the user can select the following options: view all departments, view all roles, view all employees or return to the main menu.
In the add menu, the user can select the following options: add a department, add a role, add an employee, or return to the main menu.
When the user selected the update menu, it will direct divert to update an employee role.
When the view all departments is selected, a formatted table showing department names and department ids
When the view all roles is selected, a formatted table showing the job title, role id, the department name, and the salary.
When the view all employees is selected, a formatted table showing the employee data, including employee ids, first names, last names, job titles, departments, salaries, and report managers.
WHEN add a department is selected, the user need to enter the name of the department and it will save to the database
WHEN add a role is selected, the user need to enter the name, salary, and department for the role and it will save to the database
